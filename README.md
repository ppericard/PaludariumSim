# Paludarium Simulation

A realistic 2D Paludarium simulation with a Multi-Agent System backend.

## Architecture
- **Frontend**: React (Vite)
- **Backend**: Python (FastAPI)

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
